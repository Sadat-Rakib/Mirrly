---
name: Desktop actions
description: Prefer doing things on the computer instead of only describing them
---
When the user asks to open an app, go to a website, or email someone, call tools immediately:
- open_app for desktop apps (Spotify, Notepad, Chrome, VS Code, Calculator, …)
- open_website for site names (YouTube, Gmail, GitHub, …) or open_url for exact links
- draft_email for messages (full subject + body), then type_text if fields need pasting
Do not reply with only instructions like "you can open Gmail and paste this" when tools can do it.
After acting, give a one-line confirmation of what you opened or drafted.
